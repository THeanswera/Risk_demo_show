# Калькулятор индивидуального пожарного риска (ИПР)

Веб-приложение на Streamlit для расчёта индивидуального пожарного риска по **Методике определения расчётных величин пожарного риска** (Приказ МЧС России от 14.11.2022 № 1140).

## Реализованные формулы

| Формула | Описание |
|---------|----------|
| (1) | Условие соответствия: R ≤ R_норм = 10⁻⁶ год⁻¹ |
| (2) | R = max{R₁, …, Rᵢ, …, Rₖ} — расчётная величина ИПР |
| (3) | Rᵢ = maxⱼ{Rᵢ,ⱼ} — ИПР по сценарию |
| (4) | Rᵢ,ⱼ = Qп,ᵢ · (1−Kап,ᵢ) · Pпр,ᵢ · (1−Pэ,ᵢ,ⱼ) · (1−Kп.з,ᵢ) — ИПР для группы |
| (5) | Pпр,ᵢ = tпр,ᵢ / 24 — вероятность присутствия |
| (6) | Pэ,ᵢ,ⱼ — вероятность эвакуации (кусочная функция) |
| (7) | Kп.з,ᵢ = 1 − (1 − Kобн·Kсоуэ)·(1 − Kобн·Kпдз) — коэффициент защиты |
| (8) | Rᵢ = 0.3·Rᵢ(откр) + 0.7·Rᵢ(закр) — учёт противопожарных дверей |

Справочные таблицы: П3.1 (частота пожара), П4.1 (tн.э), П9.1 (пожарная нагрузка).

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Структура проекта

```
app.py                  # Точка входа Streamlit
requirements.txt
core/
  constants.py          # Нормативные константы
  formulas.py           # Формулы (4)–(8)
  tables.py             # Таблицы П3.1, П4.1
  compute.py            # Основная логика compute_all()
  t_block.py            # Таблица П9.1 + расчёт времени блокирования tбл (Приложение 6)
ui/
  sidebar.py            # Боковая панель, расчёт tбл
  scenarios.py          # Шаг 1: сценарии
  groups.py             # Шаг 2: группы
  doors.py              # Шаг 3: противопожарные двери
  results.py            # Шаг 4: результаты и диагностика
  reference.py          # Справочные таблицы и формулы
export/
  report_docx.py        # Генерация Word-отчёта (python-docx)
visualization/
  gauge.py              # HTML-шкала логарифмического риска
utils/
  helpers.py            # clamp, safe_float, форматирование
tests/
  test_formulas.py      # Unit-тесты формул (5)–(8)
  test_compute.py       # Интеграционные тесты compute_all()
```
